project model 说明

参数名 值类型 取值范围 取值例子 说明
uuid string len=16 由创建者uuid+随机8位字符组成
name string len<=128 项目名称
assign string len=8 项目负责人
status_uuid string len=8 项目状态uuid
status_category string 项目状态分类
is_pin bool 是否置顶
status int 项目状态，1：正常；2：删除 (兼容移动端，目前接口只返回status=1的数据)
is_open_email_notify bool 是否开通邮件通知
announcement string len<=512 项目公告
deadline int64 截止时间，与1970的秒差
task_update_time int64 项目下任务更新时间(单位秒)

API 说明

1. 添加项目

URL

https://api.ones.ai/v1/team/:teamUUID/projects/batch_add

HTTP Method

POST

调用权限

administer_do

传值方式

JSON

参数列表

参数名 值类型 是否可以空 取值范围 取值例子 说明
projects array 否 批量添加项目数组
 uuid string 否 len=16 由创建者uuid+随机8位字符组成
 name string 否 len<=128 项目名称
 announcement string 是 len<=512 项目公告
 deadline int64 是 截止时间，与1970的秒差

项目成员域类型

type 值类型
single_user 用户
group 用户组
department 部门
everyone 团队内所有用户

返回JSON

参数名 值类型 说明
projects array 添加成功project,见上方model说明
bad_projects array 添加失败的project
 uuid string 添加失败的project的id
 code int 添加失败错误码
 desc string 添加失败提示
server_udpate_stamp int64 更新时间戳

返回的HTTP status code

状态码 说明
200 操作成功
500 接口错误
801 参数无效
832 超出数量限制

4. 根据项目id获取项目列表

URL

https://api.ones.ai/v1/team/:teamUUID/projects/info

HTTP Method

GET

调用权限

需要 查看团队项目列表 权限

参数列表

参数名 是否必须 值类型 取值范围 默认值 取值例子 说明
ids T string ojytvgw3876gdkut,juyg5oi87tgdwet6 多个使用逗号隔开,单个uuid长度为16，并且不能重复

返回的HTTP status code

状态码 说明

400 请求参数无效

参数名 值类型 取值范围 取值例子 说明
projects array 项目数组,参考上面project model
errors array 获取失败的项目列表
  uuid string 项目uuid
  code int 错误码
  desc string 错误描述

code 可能取值

code 说明
403 无访问权限
609 项目uuid不存在
819 没有操作权限

11. 添加项目时导入模板接口

URL

HTTP Method

POST

调用权限

manage_project

传值方式

JSON

参数列表

参数名 值类型 是否可以空 取值范围 取值例子 说明
project object 否 添加项目
 uuid string 否 len=16 由创建者uuid+随机8位字符组成
 name string 否 len<=128 项目名称
 announcement string 是 len<=512 项目公告
 deadline int64 是 截止时间，与1970的秒差
issue_types array 是 任务类型的UUID数组
copy_from map 是 key:project_uuid, value:issue_type_uuid数组
members array 是 “项目成员”角色里的成员

返回JSON

参数名 值类型 说明
project object 更新成功project,见上方model说明
server_update_stamp int64 issue_type类型的数据的更新时间

返回的HTTP status code

状态码 说明
200 操作成功
500 接口错误
801 参数无效
832 超出数量限制
